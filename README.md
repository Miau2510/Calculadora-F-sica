# ⚛️ Calculadora de Física em Python

Este é um script desenvolvido em Python para automatizar cálculos de física, focado inicialmente em **Aceleração Centrípeta**. O objetivo foi criar uma ferramenta prática para validar exercícios e facilitar o estudo de movimentos circulares.

## 🚀 Funcionalidades

- **Cálculo de Aceleração Centrípeta:** Calcula a aceleração com base na velocidade linear (m/s) e no raio da trajetória.
- **Conversão Automática:** O programa aceita o raio em **Metros** ou **Quilómetros**, realizando a conversão interna necessária para o Sistema Internacional (SI).
- **Interface de Terminal:** Menu interativo com limpeza de ecrã automática para melhor legibilidade.

## 🛠️ Como usar

1. Certifica-te de que tens o Python instalado no teu sistema (Linux/Debian).
2. Clona este repositório ou descarrega o ficheiro `calculos_fisica.py`.
3. Executa o programa no terminal:
   ```bash
   python3 calculos_fisica.py
