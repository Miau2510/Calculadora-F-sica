# ⚛️ Physics Calculations with Python

Este repositório contém uma ferramenta em Python desenvolvida para automatizar cálculos de física mecânica. O projeto foca na simplicidade, leveza e precisão, servindo como um assistente eficiente para a resolução de exercícios de cinemática e dinâmica.
🚀 Módulos Implementados
1. Aceleração Centrípeta

    Calcula a aceleração necessária para manter um corpo em trajetória circular.

    Conversão Automática: Aceita a entrada do raio tanto em metros como em quilómetros, realizando a conversão interna para o Sistema Internacional (SI).

2. Transmissão de Movimento (Bicicletas/Engrenagens)

    Calcula a relação de frequências entre duas engrenagens (como coroa e pinhão) conectadas por uma corrente.

    Baseado na igualdade da velocidade linear nas extremidades:
    f1​⋅R1​=f2​⋅R2​

    Validado com sucesso em exercícios práticos de cadência de pedalada.

# 🛠️ Filosofia do Projeto

   Clean Code: Interface minimalista via terminal, sem dependências externas ou poluição visual, facilitando o uso em qualquer ambiente.

   Robustez: Implementação de lógica condicional (if/elif/else) para garantir que o utilizador selecione apenas opções válidas.

   UX Simples: Utiliza pausas (input()) e limpeza de ecrã (os.system) para manter o fluxo de trabalho organizado no terminal.

# 📖 Como Executar

Certifique-se de que tens o Python 3 instalado no teu sistema (Linux, Windows ou MacOS).

   Clone o repositório ou descarregue o ficheiro calculos_fisica.py.

   No terminal, execute:
    

    python3 calculos_fisica.py
